This project aims to create a distributed security camera system.

Folder Structure:
- Planning folder will contain requirements, specifications, architecture
  diagrams, napkin sketches of circuits, research, etc.
- Implementation folder will hold source code, schematics, board layouts, etc.
- Testing folder will hold test code, test plans, test result documents, etc.
- DeploymentAndUse folder will hold instructions on provisioning and deploying
  a working system.
